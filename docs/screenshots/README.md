# Screenshots Directory

This directory should contain screenshots of the application for the README.md file.

## Required Screenshots:

1. **products_screen.png** - Main products list screen
2. **add_product.png** - Add product dialog
3. **location_screen.png** - GPS location screen

## How to Add Screenshots:

1. Run the app on an Android emulator or device
2. Take screenshots of each screen
3. Save them with the names listed above
4. Place them in this directory
5. The README.md will automatically display them

## Tips:

- Use PNG format for better quality
- Recommended size: 1080x1920 or device screen resolution
- Make sure the screenshots show the app in action
- Include both light and dark mode if applicable
